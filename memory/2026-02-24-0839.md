### 🔥 Ne İşe Yarıyor?
Scrapling, tek bir HTTP request'ten tam ölçekli crawl'a kadar her şeyi handle eden **adaptive web scraping framework'ü**. Python 3.10+ için yazılmış.

### ⚡ Neden Farklı?

**1. Adaptive Parsing (En killer özellik!)**
- Site yapısı değiştiğinde elementleri otomatik yeniden buluyor
- `auto_save=True` ile ilk çalıştırmada elementi kaydediyor, sonra `adaptive=True` ile site değişse bile buluyor
- Benchmark'lara göre AutoScraper'dan **5x daha hızlı** bu işi yapıyor

**2. Built-in Anti-Bot Bypass**
- Cloudflare Turnstile/Interstitial'i otomatik bypass ediyor
- TLS fingerprint impersonation (Chrome, Firefox gibi)
- StealthyFetcher ile "görünmez" fetch yapabiliyor

**3. Full Spider Framework**
- Scrapy benzeri API ama daha modern ve basit
- Concurrent crawling, per-domain throttling
- **Pause/Resume** - Ctrl+C ile durdur, devam ettiğinde kaldığı yerden devam et
- Streaming mode - real-time stats ile crawl sonuçlarını anında al

**4. Performance**
- Benchmark'lara göre:
  - Parsel/Scrapy: **1.01x** (neredeyse eşit)
  - BeautifulSoup: **767x daha hızlı**! 😱
  - Selectolax: **41x daha hızlı**

**5. MCP Server Entegrasyonu**
- Claude/Cursor gibi AI tool'larıyla kullanım için built-in MCP server
- AI'a göndermeden önce Scrapling ile hedefli content çıkarıyor → token tasarrufu

### 🎯 Kullanım Senaryoları

```python
# Basit fetch (stealthy)
from scrapling.fetchers import StealthyFetcher
page = StealthyFetcher.fetch('https://example.com', headless=True)
data = page.css('.product', auto_save=True)

# Full spider (Scrapy gibi)
from scrapling.spiders import Spider
class MySpider(Spider):
    name = "demo"
    start_urls = ["https://example.com/"]
    async def parse(self, response):
        for item in response.css('.product'):
            yield {"title": item.css('h2::text').get()}
```
